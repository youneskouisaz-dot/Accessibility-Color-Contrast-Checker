# Project Brief

## Project Title
Accessibility Color Contrast Checker

## Topic
This project focuses on digital accessibility. The goal is to help users determine whether the colors they choose for text and backgrounds provide enough contrast to be easily read.

## Artifact
The artifact is a Python program that accepts two hexadecimal color values, calculates their contrast ratio, and determines whether they meet the Web Content Accessibility Guidelines (WCAG).

## Technical Concepts
This project uses:
- Python programming
- Functions
- User input
- Conditional statements
- Mathematical calculations
- Hexadecimal color values

## Social and Ethical Question
Many people with visual impairments or color blindness struggle to read websites with poor color contrast. Developers have a responsibility to make digital content accessible to everyone. This project demonstrates how programming can help improve accessibility.

## Conclusion
This project combines programming with accessibility by creating a simple tool that helps users evaluate color combinations according to accessibility standards.
